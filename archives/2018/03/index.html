进程的基本状态:

三种基本状态:
    就绪状态:进程具备运行条件，等待系统分配处理器以便行。
    等待状态:又称为阻塞(blocked)态或睡眠(sleep)态，指进程不具备运行条件，正在等待某个事件的完成。
    运行状态:进程占有处理器正在运行。

    需要远程ssh

    ssh -p xxx(你设定的端口) xxx(用户名)@xxx.xxx.xxx.xxx(你的服务器ip)

  • 需要服务器管理防火墙开启UDP 53端口(打开方法自行百度)

    可以用xhell连接并传送vpnserver.tar.gz文件到服务器

  • 运行下面命令

    $: yum install gcc gcc-c++ make tar -y

    $: cd /root

    $: tar -zxvf vpnserver

  • 解压完毕(then)运行源码

    $: cd vpnserver

    $: ./.install.sh

  • 然后根据提示输入1 回车

  • then

    $: ./vpnserver start

    $: ./vpncmd

  • 选1两次回车

  • 此时显示

    VPN server>

    使用openvpn在服务器端开启服务

  • 新建连接

  • 点击连接 输入密码，确认密码
