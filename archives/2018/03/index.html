校园网绕过web认证

2018-03-12

  • 注意

  • 目前这样的适用(其他的我还没试过)

  • 1 有一台用作转发的服务器

    百度阿里(有梦想的去微软试用机)

    需要远程ssh

ssh -p xxx(你设定的端口) xxx(用户名)@xxx.xxx.xxx.xxx(你的服务器ip)

  • 需要服务器管理防火墙开启UDP 53端口(打开方法自行百度)

    可以用xhell连接并传送vpnserver.tar.gz文件到服务器

  • 运行下面命令

    $: yum install gcc gcc-c++ make tar -y

    $: cd /root

    $: tar -zxvf vpnserver

  • 解压完毕(then)运行源码

    $: cd vpnserver

    $: ./.install.sh

  • 然后根据提示输入1 回车

  • then

    $: ./vpnserver start

    $: ./vpncmd

  • 选1两次回车

  • 此时显示

    VPN server>

使用openvpn在服务器端开启服务

  • 你的电脑安装管理工具

  • 新建连接

  • 点击连接 输入密码，确认密码

  • 选择 VPN 的其他高级配置
  • 否
  • 确定

  • 生成的文件把后缀为_l3的.ovpn文件解压出来既可

  • 并更改”.ovpn”文件中的服务器地址和转发端口(改为你的 服务器ip和53端口)
  • 原1 -> 更改为2

    1: remote vpnxxxxxxx.v4.sedns.cn 1194

    2: remote xxx.xxx.xxx.xxx 53

  • xxx.xxx.xxx.xxx 为你的服务器ip

  • 保存文件

    连接(先连接到你的校园网)

    安装openvpn
    用安装好的openvpn打开之前生成并配置好的 “.ovpn”文件 then 接受 之后输入用户的账号和密码即可

    连接完成

  • 可以再检查一下ip是否为你服务器的ip

  • 完成！
